# vim的基本使用

## 模式

normal模式 光标较大,不能输入内容  快捷键(ESC|ctrl+[)
insert模式 可以输入内容 快捷键，在normal模式使用(i在当前光标,a在当前光标后面)

## 光标移动(normal模式下)

K 往上移动
J 往下移动
H 往左移动
L 往右移动

练习

```js
const a = 1
const b = 2
```

## 终端中使用vim退出

:q
:q!(强制退出)

## 掌握和行相关的命令

### 移动

+ 行首 0
+ 行首 ^(shift+6)移动到第一个不是blank字符的位置 blank 空格、tab、换行、回车等
+ 行尾 $
+ 行尾 g_到本行最后一个不是blank字符的位置
+ 最后一行 shift+g

### 插入

+ 行首 I
+ 行尾 A
+ 行前 O
+ 行后 o

### 复制

+ 复制当前行 yy
+ 删除当前行 dd
+ 粘贴 p

## vim语法

**操作符+动作(范围)**

### 操作符

+ 删除 d
+ 删除后进入insert模式 c
+ 复制 y

### 基于单词、字符串的移动

+ e 移动到单词的结尾(括号什么的算一个单词，E会以字符串为单位，跳过括号)
+ b 移动到上一个单词的开头
+ w 移动到单词的开头
+ ge 移动到上一个单词的结尾

### 组合

+ cw 删除当前单词
+ ea 在当前单词的结尾添加
