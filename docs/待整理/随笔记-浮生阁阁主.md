## HTML a标签 href属性
1. 阻止a标签跳转，点击执行函数
```html
<a href="javascript:;" onclick="doSomething()">执行函数</a>
```
2. href属性改变页面href
```html
<a href="/user/self">主页</a>
```
3. href属性链接到平级页面
```html
<a href="/user/../cart/">购物车</a>
```
4. 链接到页面茅点
```html
<a href="#">顶部</a>
```
